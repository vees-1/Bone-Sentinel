# Bone-Sentinel
Built a CNN-based binary classification system using TensorFlow/Keras to detect fractures in musculoskeletal X-ray images from the MURA dataset. Trained and optimized the model, evaluating performance using validation accuracy and F1-score for reliable diagnostic support.
